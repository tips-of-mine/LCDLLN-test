## Contenu du ticket

# M38.3_Dynamic_lighting_streetlamps_+_windows_+_torches

## Objectif
Lumières dynamiques (torches, lampes) s'allument la nuit, s'éteignent le jour.

## Dépendances
- M38.1
- M03.2

## Livrables
- Point lights activés/désactivés par time of day
- Streetlamp entities (auto-trigger)
- Window light emissive (shader)

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Point lights: position, radius, color, intensity
- Trigger: activate si time 18:00-6:00 (night)
- Fade in/out: smooth transition (1min)
- Emissive: boost albedo intensity at night

## Étapes d'implémentation
1. Tag lights "auto_night_light"
2. Update: check time of day
3. If night && !active: fade in intensity (0->1 over 60s)
4. If day && active: fade out
5. Deferred lighting: add point lights (M3.2 extend)
6. Windows: material emissive multiplier (time-based)
7. Torches: particle FX + point light

## Definition of Done (DoD)

- [ ] Aucun dossier racine non autorisé créé
- [ ] Rapport final: fichiers modifiés + commandes + résultats + DoD

## Notes / pièges à éviter
- Limit concurrent lights (100-200 max visible)
- Cluster lights (deferred rendering)
