# M22.4 — Ticket system: client connects to shard only with Master-issued ticket

**Status:** Closed

## Contenu du ticket (référence)

- **Objectif** : Empêcher toute connexion directe à un shard sans ticket temporaire émis par le Master.
- **Livrables** : Ticket struct (ticket_id 128-bit, account_id, target_shard_id, expires_at, signature/HMAC), opcodes REQUEST_SHARD_TICKET / SHARD_TICKET_RESPONSE, validation ticket côté shard.
- **Spec** : Ticket à usage unique (consume on first use), expiration courte, v1 HMAC secret partagé.
