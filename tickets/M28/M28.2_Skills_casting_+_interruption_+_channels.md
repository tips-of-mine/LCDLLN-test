# M28.2_Skills_casting_+_interruption_+_channels

## Objectif
Implémenter casting avec barre de progression, interruption, et channeling.

## Dépendances
- M28.1
- M16.1

## Livrables
- Cast state machine (idle, casting, channeling, cooldown)
- Interruption conditions (damage, stun, movement)
- Channel ticks (DoT, HoT)

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Casting: timer [0..castTime], frozen movement optionnel
- Interruption: damage >X%, stun, knockback, death
- Channeling: tick every N sec, early cancel possible
- GCD (global cooldown) 1-1.5s typique

## Étapes d'implémentation
1. State machine: idle -> casting -> (success|interrupted)
2. Casting: update timer, check interruption conditions
3. On completion: apply skill effects, start cooldown
4. On interrupt: refund partial mana (50%?), no effect
5. Channeling: schedule ticks, allow early cancel
6. GCD: block all skills for 1.5s after any cast

- [ ] Build Windows OK (configure + build) via presets existants

- [ ] Aucun dossier racine non autorisé créé

## Notes / pièges à éviter
- Instant casts (castTime=0) skip casting state
- Client prediction cast start, serveur valide
