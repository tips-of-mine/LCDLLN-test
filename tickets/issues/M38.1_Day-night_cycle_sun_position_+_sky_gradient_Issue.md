## Ticket source

# M38.1_Day-night_cycle_sun_position_+_sky_gradient

## Objectif
Cycle jour/nuit avec soleil/lune animés, sky gradient, éclairage dynamique.

## Dépendances
- M04.1
- M05.2

## Livrables
- Time of day (0-24h, speed configurable)
- Sun/moon position (celestial mechanics)
- Sky gradient (Rayleigh scattering approximation)
- Directional light rotation + color/intensity

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Time: 0-24h, update dt * timeScale (1 = realtime, 60 = 1h realtime = 1day ingame)
- Sun position: latitude/longitude -> azimuth/elevation
- Sky colors: dawn=orange, noon=blue, dusk=red, night=dark blue
- Light intensity: day=1.0, night=0.1 (moonlight)

## Étapes d'implémentation
1. Time system: current time (float 0-24), advance each frame
2. Calculate sun angle: elevation = sin(time * pi / 12 - pi/2)
3. Sun direction: rotate directional light
4. Sky shader: gradient top->horizon (Rayleigh scatter approximation)
5. Light color: lerp white->orange (sunrise/set), white (noon)
6. Ambient light: scale by sun elevation
7. Moon: opposite side (sun + 12h)

## Rapport final

### 1) Fichiers

**Créés :**
- `engine/render/DayNightCycle.h` — Classe `DayNightCycle` : horloge 0-24 h, avancement par frame (timeScale configurable), calcul de `sunElevation` = sin(t·π/12 − π/2) conformément au ticket, direction lumière, couleur/intensité interpolée (nuit/aube/midi/crépuscule), couleur ambiante avec plancher anti-nuit-noire, gradient sky (zenith + horizon), position lune = soleil + 12 h.
- `engine/render/DayNightCycle.cpp` — Implémentation avec LOG_INFO Init/SetTime et LOG_TRACE dans Advance.
- `game/data/shaders/sky.vert` — Vertex shader plein-écran (triangle géométrique).
- `game/data/shaders/sky.frag` — Fragment shader sky : gradient zenith→horizon par approximation Rayleigh (pow blend), disque soleil + halo Mie simplifié.

**Modifiés :**
- `engine/Engine.h` — Include `DayNightCycle.h`, membre `m_dayNight` (engine::render::DayNightCycle).
- `engine/Engine.cpp`
  - `LoadZoneProbeAssets()` : initialise `m_dayNight` avec paramètres JSON (`world.day_night.initial_time`, `world.day_night.time_scale`).
  - `Update()` : appel de `m_dayNight.Advance(dt)` puis propagation vers `m_zoneAtmosphere` (sunDirection, sunColor, ambientColor). Le chemin de rendu existant (LightingPass::LightParams) lit `m_zoneAtmosphere` sans modification.
- `CMakeLists.txt` — Ajout de `engine/render/DayNightCycle.cpp` dans les sources.

**Supprimés :** aucun.
