# M25.2_Animation_GPU_skinning_+_vertex_shader

## Objectif
Appliquer pose squelettique aux vertices via vertex shader (matrix palette skinning).

## Dépendances
- M25.1
- M03.1

## Livrables
- Shader skinning (VS: apply joint matrices)
- Uniform buffer joint matrices (max 256 joints)
- Mesh format étendu (weights, indices)

## Structure & chemins (verrouillé)
- **Code moteur** : uniquement sous `/engine/...`
- **Code jeu** : uniquement sous `/game/...`
- **Contenu** (textures/meshes/json/audio/shaders) : uniquement sous `/game/data/...`
- **Outils offline** : uniquement sous `/tools/...`
- **Interdit** : créer des dossiers racine du type `/assets`, `/textures`, `/data`, `/shaders`, etc.
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json). Pas de chemins absolus.


## Spécification technique
- Max 4 influences par vertex (skinIndices, skinWeights)
- Joint matrices 4x4 (ou dual-quaternions phase 2)
- Upload matrices par instance si multi-character

## Étapes d'implémentation
1. Étendre mesh format: attributs skinIndices (uvec4), skinWeights (vec4)
2. Calculer pose world-space depuis local-space (traverse hiérarchie)
3. Upload joint matrices[256] en UBO
4. Vertex shader: finalPos = Σ(jointMatrix[idx[i]] * localPos * weight[i])
5. Appliquer aussi aux normales/tangentes
