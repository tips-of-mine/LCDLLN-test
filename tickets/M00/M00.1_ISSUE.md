# M00.1 — Core: logging, time, config

**Status:** Open

## Ticket (original)

### Objectif
Mettre en place un socle Core réutilisable (engine/game/tools) : logs robustes, timing frame, config runtime + overrides CLI.

### Dépendances
- Aucune

### Livrables
- `/engine/core/Log.h|.cpp`, macros LOG_* (subsystem + niveau)
- `/engine/core/Time.h|.cpp` (delta, fps, frameIndex, NowTicks)
- `/engine/core/Config.h|.cpp` (load json/ini, Get*, overrides `--key=value`)

### Structure & chemins (verrouillé)
- **Code moteur** : uniquement sous `/engine/...`
- **Contenu** (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- **Outils offline** : uniquement sous `/tools/...`
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json).  
  Exemple : charger `textures/brick_albedo.png` depuis `${paths.content}/textures/brick_albedo.png`.
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

### Spécification technique
- Format log: `[YYYY-MM-DD HH:MM:SS.mmm][T:threadId][LEVEL][Subsystem] message`.
- Logs thread-safe, console + fichier, filtrage par niveau.
- Time basé sur horloge monotone; FPS moyenne glissante (ex: 120 frames).
- Config: valeurs par défaut si fichier absent; overrides CLI prioritaires.

### Étapes d'implémentation
1. Implémenter `Log::Init/Shutdown`, writer console + file, mutex ou lock-free simple.
2. Ajouter macros `LOG_INFO(Render, ...)` etc., + `LOG_FATAL` (log + abort).
3. Implémenter Time (BeginFrame/EndFrame), delta clamp, fps window.
4. Implémenter Config (parser), getters typés, merge defaults + file + CLI.
5. Ajouter petit exemple au boot (log version, fps 1/sec).

### Critères de validation (DoD)
- [ ] Logs depuis main + threads jobs (1000 logs) sans corruption.
- [ ] Fichier log écrit et flush en debug.
- [ ] Delta jamais négatif; FPS plausible; frameIndex incrémente.
- [ ] Config charge défaut si absent et applique overrides CLI.

### Notes / pièges à éviter
- Éviter allocations dans le hot path de log (buffer fixe si possible).
- Time: utiliser `steady_clock` / QPC, pas `system_clock`.
