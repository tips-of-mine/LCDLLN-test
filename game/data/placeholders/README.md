# assets/placeholders/

🇫🇷 FRANÇAIS
## Pourquoi ce dossier existe
Il fournit des images de secours pour éviter les erreurs d’affichage
ou les crashs lorsque des ressources sont manquantes.

## Ce qu’il doit contenir
- Image de chargement.
- Image “ressource manquante”.
- Avatar vide par défaut.

🇬🇧 ENGLISH
## Why this folder exists
It provides fallback images to avoid UI issues or crashes
when assets are missing.

## What it must contain
- Loading image.
- Missing resource placeholder.
- Default empty avatar.
