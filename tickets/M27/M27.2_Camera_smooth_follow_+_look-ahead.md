# M27.2_Camera_smooth_follow_+_look-ahead

## Objectif
Améliorer feeling caméra avec follow smooth et anticipation direction mouvement.

## Dépendances
- M27.1
- M26.1

## Livrables
- Spring-damper follow
- Look-ahead offset basé vélocité
- Transition camera states (combat/exploration)

## Spécification technique
- Spring-damper: stiffness/damping configurable
- Look-ahead: offset = velocity * 0.5-1.0s
- Combat mode: resserrer distance, lock target optionnel

## Étapes d'implémentation
1. Remplacer lerp par spring-damper (smoother)
2. Calculer look-ahead offset: character velocity projected
3. Target position = character pos + look-ahead + up offset
4. Mode combat: override distance/pitch
5. Exposer params config camera

## Notes / pièges à éviter
- Tuning spring critical: trop soft = laggy, trop stiff = jittery
- Clamp look-ahead offset (max 2-3m)
