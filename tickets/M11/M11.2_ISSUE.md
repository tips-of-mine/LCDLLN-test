# M11.2 — Zone Builder: chunking + outputs structure

**Status:** Closed

## Ticket (original)

### Objectif
Découper une zone en chunks et écrire outputs `build/zone_x/chunks/chunk_i_j/*`.

### Dépendances
- M11.1

### Livrables
- Chunk assignation
- chunk.meta
- instances.bin

### Spécification technique
- Chunk coord = floor(x/256), floor(z/256).

### Étapes d'implémentation
1. Découper instances par chunk.
2. Écrire chunk.meta (bounds, flags).
3. Écrire instances.bin (transform + assetId + flags).
4. Écrire zone.meta (liste chunks).

### Critères de validation (DoD)
- [ ] Runtime peut charger chunk.meta + instances.bin et afficher placeholders.

### Notes
- Prévoir neighbor list optionnel si objets bord.
