# M37.2_Water_foam_+_shore_waves_+_caustics

## Objectif
Détails eau: foam edges, waves shore, caustics underwater.

## Dépendances
- M37.1

## Livrables
- Foam texture (depth-based)
- Shore waves (vertex animation)
- Caustics (projected texture)

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Foam: depth difference water/terrain < threshold (0.5m)
- Waves: sine wave vertex displacement (amplitude 0.2m)
- Caustics: animated texture projected from sun

## Étapes d'implémentation
1. Foam: depth = sceneDepth - waterDepth
2. If depth < 0.5m: blend foam texture
3. Shore waves: vertex shader Y += sin(time + posX)
4. Caustics: sample animated texture (scroll UV)
5. Project caustics on terrain under water (shader)
6. Modulate lighting avec caustics

## Rapport final (M37.2)

### 1) FICHIERS

**Modifiés :**
- `engine/render/WaterPass.h`
  - `WaterParams` étendu de 96 → **120 bytes** : +6 champs M37.2 (`waveAmplitude`, `foamThreshold`, `causticsScale`, `causticsIntensity`, `nearZ`, `farZ`)
  - `Record()` : nouveau paramètre `ResourceId idDepth` (scène depth pour le foam)
  - Nouveau membre privé `VkSampler m_depthSampler` (NEAREST, clamp)

- `engine/render/WaterPass.cpp`
  - Sampler 4a (linear, existant) renommé ; nouveau **4b depth sampler** (NEAREST filter, clamp-to-edge)
  - Descriptor set layout : **3 bindings** (0=reflection, 1=refraction, 2=sceneDepth)
  - Pool size : 3 × maxFrames (était 2)
  - `Record()` : mise à jour dynamique du binding 2 avec la vue depth avant le rendu
  - `Destroy()` : destruction de `m_depthSampler`

- `game/data/shaders/water.vert`
  - Bloc push_constant étendu aux 12 champs (miroir de `WaterParams`)
  - **Shore wave vertex animation** : `Y += waveAmplitude * (sin(time * 1.4 + x * 0.8)*0.6 + sin(time * 0.9 + z * 0.6)*0.4)`

- `game/data/shaders/water.frag`
  - Bloc push_constant étendu
  - Binding 2 : `uniform sampler2D sceneDepthTex`
  - Fonction `LineariseDepth(ndcDepth)` : linearisation perspective (`n*f / (f - d*(f-n))`)
  - **Foam depth-based** : `depthDiff = linearScene - linearWater` ; si `< foamThreshold` → blend procedural foam (sine noise, kFoamColor = blanc cassé)
  - **Caustics proceduraux** : 3 fronts d'interférence animés (`sin*sin`), power 3 pour sharpness, injecté dans refractColor
