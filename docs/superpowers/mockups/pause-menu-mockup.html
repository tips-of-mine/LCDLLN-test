À savoir sur le rendu réel (ImGui / Vulkan) — la police en jeu est Windlass (décorative, capitales) ; ici j'utilise un serif système pour l'approcher. Les dégradés et halos de la variante C sont possibles en ImGui mais via ImDrawList custom (comme le toggle auth) ; A et B sont réalisables avec les helpers existants (DrawAuthButtonPrimary, DrawAuthButtonGhost, DrawAuthButtonDanger) presque tels quels.

Le centrage est corrigé dans les trois cas : largeur de bouton fixe, posée à X = (largeur_contenu − largeur_bouton) / 2, titre centré sur la zone de contenu réelle (padding inclus).
