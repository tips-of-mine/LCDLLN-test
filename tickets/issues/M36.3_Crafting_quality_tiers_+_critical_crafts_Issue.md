## Contenu du ticket

# M36.3_Crafting_quality_tiers_+_critical_crafts

## Objectif
Système qualité craft (normal, rare, epic) avec criticals et stat bonuses.

## Dépendances
- M36.2

## Livrables
- Quality tiers: normal, uncommon, rare, epic
- Critical craft chance (skill-based)
- Stat scaling par qualité (+5% stats per tier)
- UI: quality indicator (border color)

## Spécification technique
- Base quality: normal (white)
- Critical craft: roll chance (skill/300 * 10% = max 10%)
- Crit result: +1 quality tier (normal->uncommon, rare->epic)
- Stat bonus: multiply base stats by tier multiplier
- Tier colors: white, green, blue, purple

## Étapes d'implémentation
1. Item template: base_stats, quality_tier
2. On craft: roll crit chance (random < skill/300 * 0.1)
3. If crit: upgrade quality tier +1
4. Calculate final stats: base * tier_multiplier
5. Item instance: store quality in DB
6. UI: color border by quality
7. Tooltip: show quality tier name

## Definition of Done (DoD)
- [ ] Build Windows OK (configure + build) via presets existants
- [ ] Exécutable lancé (smoke test) et pas de crash immédiat
- [ ] Aucun dossier racine non autorisé créé
- [ ] Rapport final: fichiers modifiés + commandes + résultats + DoD

## Notes / pièges à éviter
- Cap quality (epic max, pas legendary via craft)
- Transmute recipes: upgrade quality (phase 2)
