# M26.2_Character_controller_jump_+_air_control

**Status:** Closed

## Objectif
Ajouter saut avec impulsion verticale et contrôle air limité.

## Dépendances
- M26.1

## Livrables
- Jump impulse
- Air control (reduced)
- Coyote time + jump buffer

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Jump velocity ~8-10 m/s (atteint 2-3m hauteur)
- Air control 50% de ground control
- Coyote time 0.1s (peut jump juste après quitter sol)
- Jump buffer 0.1s (input anticipé avant atterrissage)

## Étapes d'implémentation
1. Input jump: si isGrounded, apply velocity.y = jumpSpeed
2. Air control: reduce acceleration multiplier
3. Coyote time: track time since left ground
4. Jump buffer: track time since jump input
5. Allow jump si coyote || buffer valid
