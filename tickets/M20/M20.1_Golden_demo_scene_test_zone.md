# M10.1 — Golden Demo — scène de référence + mode démo

> **Type**: Ticket (Engineering)  
> **Priority**: High  
> **Owner**: Codex  
> **Status**: TODO  
> **Created**: 2026-02-08

## Objectif
Créer une scène de référence (“golden demo”) qui sert de base de validation visuelle/perf pour tous les futurs changements.

La démo doit:
- démarrer directement dans une zone test (4 km) avec un “player/camera” contrôlable,
- afficher les overlays debug utiles (rings, budgets, GPU timings si dispo),
- fournir des toggles CLI/config pour activer/désactiver les features majeures (TAA/SSAO/Bloom/CSM/IBL).

**Important**: l’objectif n’est pas de créer du contenu artistique final, mais un banc de test stable et reproductible.

## Structure & chemins (verrouillé)
- **Code moteur** : uniquement sous `/engine/...`
- **Code jeu** : uniquement sous `/game/...`
- **Contenu** (textures/meshes/json/audio/shaders) : uniquement sous `/game/data/...`
- **Outils offline** : uniquement sous `/tools/...`
- **Interdit** : créer des dossiers racine du type `/assets`, `/textures`, `/data`, `/shaders`, etc.
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json). Pas de chemins absolus.


## Livrables
- `game/DemoApp.*` (ou équivalent) : entrée démo/scene test
- Ajout d’un mode `--app.mode=demo` (ou `demo=true`)
- Une scène minimaliste générée en code (quelques primitives) ET/OU une scène data-driven dans `game/data/demo/`
- Overlay texte minimal (logs ou debug draw) indiquant le preset et les toggles actifs

## Validation
- Lancement: `--app.mode=demo` affiche une scène stable
- Caméra se déplace + tourne
- Toggles: activer/désactiver au moins 4 features via CLI/config
- Aucun crash lors d’un resize fenêtre

## Commandes (Windows)
- Configure: `cmake --preset win-ninja-vcpkg`
- Build: `cmake --build --preset win-build`
- Run: `./build/engine_app.exe <args>`

## Definition of Done (DoD)
- [ ] Build Windows OK (configure + build) via presets existants
- [ ] Exécutable lancé (smoke test) et pas de crash immédiat
- [ ] Aucun dossier racine non autorisé créé
- [ ] Rapport final: fichiers modifiés + commandes + résultats + DoD
