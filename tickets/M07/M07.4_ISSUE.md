# M07.4 — TAA pass: reprojection + clamp anti-ghost

**Status:** Closed

## Ticket (original)

### Objectif
Implémenter pass TAA : reproject history via velocity, clamp neighborhood, blend.

### Dépendances
- M07.2
- M07.3

### Livrables
- Pass TAA: read SceneColor_LDR + HistoryPrev + Velocity + Depth
- Output HistoryNext (et/ou TAA_Output)

### Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu : uniquement sous `/game/data/...`
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

### Spécification technique
- Clamp history dans min/max neighborhood 3x3; alpha ~0.9.

### Étapes d'implémentation
1. Créer pass TAA dans FG.
2. Shader: uvHistory = uv - velocity; sample history.
3. Clamp via neighborhood min/max current.
4. Blend out = lerp(current, historyClamped, alpha).

### Critères de validation (DoD)
- [ ] Image plus stable, scintillement réduit, ghosting modéré.

### Notes / pièges à éviter
- Traiter uvHistory hors écran -> fallback current.
