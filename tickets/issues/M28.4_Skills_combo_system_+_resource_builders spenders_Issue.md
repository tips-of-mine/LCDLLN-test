## Spécification technique
- Combo points: 0-5 typique, stack per target optionnel
- Builder: cost mana, generate 1-2 combo points
- Spender: cost combo points, no mana, high damage
- Decay: reset combo on target death ou timeout (30s)

## Étapes d'implémentation
1. Étendre PlayerStats: comboPoints, maxCombo
2. Skill effects: "generateCombo": 1 ou "consumeCombo": true
3. Builder skill: on hit, add combo points
4. Spender skill: validate combo >= threshold, consume all
5. Reset combo on target change (optional) ou death
6. UI: display pips/orbs (M16)
