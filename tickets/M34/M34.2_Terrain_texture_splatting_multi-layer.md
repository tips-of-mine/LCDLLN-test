# M34.2_Terrain_texture_splatting_multi-layer

## Objectif
Appliquer multiple textures (grass, dirt, rock, snow) via splat map.

## Dépendances
- M34.1

## Livrables
- Splat map (RGBA texture, 4 layers)
- Shader blending (triplanar optionnel)
- Texture arrays (albedo, normal, ORM per layer)

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Splat map: R=grass, G=dirt, B=rock, A=snow
- 4 layers max MVP (extensible 8+ avec 2 splat maps)
- Texture tiling: 10-20m per tile
- Triplanar projection (évite stretching pentes)

## Étapes d'implémentation
1. Splat map texture (1024x1024 RGBA8)
2. Texture arrays: [grass, dirt, rock, snow] albedo/normal/ORM
3. Shader: sample splat map weights
4. Blend textures: color = Σ(tex[i] * weight[i])
5. Triplanar: sample XY, XZ, YZ planes, blend by normal
6. Tiling parameters per layer
