# assets/ui/splash/

🇫🇷 FRANÇAIS
## Pourquoi ce dossier existe
Ce dossier contient les images utilisées lors du tout premier écran affiché
au lancement de l’application, avant toute interaction utilisateur.

Il permet :
- d’afficher rapidement une interface visuelle,
- de masquer le temps d’initialisation,
- d’améliorer la perception de performance.

## Ce qu’il doit contenir
- Logo du jeu.
- Fond simple ou image de démarrage.
- Indicateur de chargement (statique ou animé).

---

🇬🇧 ENGLISH
## Why this folder exists
This folder contains images used by the very first screen displayed
when the application starts, before any user interaction.

It allows:
- immediate visual feedback,
- hiding initialization time,
- improving perceived performance.

## What it must contain
- Game logo.
- Simple background or splash image.
- Loading indicator (static or animated).
