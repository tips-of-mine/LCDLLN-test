# assets/themes/

🇫🇷 FRANÇAIS
## Pourquoi ce dossier existe
Il permet de définir l’apparence de l’interface via des thèmes interchangeables.

## Ce qu’il doit contenir
- Un dossier par thème (`default/`, `dark/`, etc.).
- Pour chaque thème :
  - `style.qss`
  - `theme.json`

🇬🇧 ENGLISH
## Why this folder exists
It allows defining the UI appearance using interchangeable themes.

## What it must contain
- One folder per theme (`default/`, `dark/`, etc.).
- For each theme:
  - `style.qss`
  - `theme.json`
