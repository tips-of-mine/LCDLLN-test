Feyhin Lokcthat — placeur interactif (repère de TON image, vue depuis le sud)

Glisse chaque bloc là où il apparaît dans ton image de concept. Le cadre EST l'image : gauche = ouest, droite = est, bas = premier plan (sud, proche), haut = fond (nord, loin). Place aussi en hauteur apparente (plus haut dans le cadre = plus loin/plus élevé). Quand c'est bon, clique « Exporter » et colle-moi le texte.

Astuce : tu peux empiler/superposer librement (ex. monument devant le massif).
