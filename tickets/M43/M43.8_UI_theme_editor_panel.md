# M43.8 — Éditeur de thèmes UI (`game/data/ui/themes/`)

**Priorité :** Normale  
**Périmètre :** `engine/editor/EditorThemePanel.cpp/.h` · `engine/editor/EditorMode.cpp/.h`  
**Dépendances :** M43.4 (ImGui intégré) · M16.5 (ThemeManager + structure `ThemeDefinition`)

---

## Objectif

Ajouter dans le mode `--editor` un panneau ImGui permettant de **lire, modifier et sauvegarder** les fichiers `theme.json` présents dans `game/data/ui/themes/` et `game/data/ui/races/`. La palette de couleurs est éditée avec des color pickers ImGui. Les modifications sont prévisualisées en direct dans les panneaux de jeu.

---

## Contexte

Chaque thème est défini par un `theme.json` contenant :

| Section | Clés |
|---------|------|
| `palette` | primary, secondary, accent, background, surface, panel, text, mutedText, border (couleurs hex `#RRGGBB`) |
| `typography` | fontFamily, uiFontFamily, baseSize, titleSize |
| `metrics` | radius, padding |
| `assets` | iconHud, iconButton, iconTooltip, fontRegular (chemins relatifs `paths.content`) |

Les thèmes disponibles sont découverts dynamiquement en scannant `ui/themes/*/theme.json` et `ui/races/*/theme.json`.

---

## Changements requis

### Nouveaux fichiers

**`engine/editor/EditorThemePanel.h`**
```cpp
#pragma once
#include "engine/client/ThemeManager.h"
#include "engine/core/Config.h"
#include <array>
#include <string>
#include <vector>

namespace engine::editor
{
    /// Thème chargé pour édition (palette en float[4] pour ImGui color picker).
    struct EditableTheme
    {
        std::string id;
        std::string displayName;
        std::string jsonPath; // chemin absolu du theme.json

        // Palette — stockée en float[4] RGBA pour ImGui::ColorEdit4
        struct ColorEntry { std::string key; float rgba[4]{ 0,0,0,1 }; };
        std::vector<ColorEntry> palette; // ordre fixe : primary, secondary, accent, background, surface, panel, text, mutedText, border

        // Typography
        std::string fontFamily;
        std::string uiFontFamily;
        int baseSize  = 14;
        int titleSize = 20;

        // Metrics
        int radius  = 8;
        int padding = 10;

        // Assets
        std::string iconHud;
        std::string iconButton;
        std::string iconTooltip;
        std::string fontRegular;
    };

    class EditorThemePanel final
    {
    public:
        /// Découvre et charge tous les thèmes depuis ui/themes/ et ui/races/.
        bool Load(const engine::core::Config& config);

        /// Sauvegarde le thème sélectionné sur disque.
        bool Save(const engine::core::Config& config);

        /// Dessine le panneau ImGui.
        void Draw();

        bool IsDirty() const { return m_dirty; }

    private:
        std::vector<EditableTheme> m_themes;
        int m_selectedThemeIndex = 0;
        bool m_dirty = false;

        void DrawThemeSelector();
        void DrawPaletteSection(EditableTheme& theme);
        void DrawTypographySection(EditableTheme& theme);
        void DrawMetricsSection(EditableTheme& theme);
        void DrawAssetsSection(EditableTheme& theme);
        void DrawPreview(const EditableTheme& theme);

        /// Convertit une couleur hex "#RRGGBB" en float[4] RGBA.
        static void HexToRgba(const std::string& hex, float out[4]);
        /// Convertit float[4] RGBA en hex "#RRGGBB".
        static std::string RgbaToHex(const float rgba[4]);
    };
}
```

**`engine/editor/EditorThemePanel.cpp`**

`Load()` :
- Scanner `{paths.content}/ui/themes/` et `{paths.content}/ui/races/` pour tous les `theme.json`
- Pour chaque fichier trouvé : parser le JSON, convertir les couleurs hex en `float[4]` via `HexToRgba`
- `LOG_INFO("[EditorTheme] Loaded {} themes", m_themes.size())`

`Save()` :
- Prendre `m_themes[m_selectedThemeIndex]`
- Reconvertir `float[4]` → hex via `RgbaToHex`
- Sérialiser et écrire le `theme.json` correspondant
- `LOG_INFO("[EditorTheme] Saved theme '{}' to {}", theme.id, theme.jsonPath)`

`Draw()` :

**Sélecteur de thème :**
```
Thème actif : [default ▼]    [Save (Ctrl+S)]
```
`ImGui::Combo` listant tous les thèmes par displayName.

**Section Palette :**
```
Palette
────────────────────────────────────
  primary     [████████] #3A6EA5
  secondary   [████████] #5A7FA8
  accent      [████████] #D9A441
  background  [████████] #10161D
  ...
```
- `ImGui::ColorEdit3("##key", rgba, ImGuiColorEditFlags_NoAlpha | ImGuiColorEditFlags_InputHex)` pour chaque couleur
- La valeur hex affichée est mise à jour en temps réel

**Section Typography :**
```
fontFamily    [Inter        ]
uiFontFamily  [Inter        ]
baseSize      [14]
titleSize     [20]
```
- `ImGui::InputText` pour les noms de police
- `ImGui::InputInt` (min 8, max 48) pour les tailles

**Section Metrics :**
```
radius   [8 ]   padding  [10]
```
- `ImGui::SliderInt` (0–32) pour radius et padding

**Section Assets :**
```
iconHud      [textures/test.texr      ]
iconButton   [textures/test.texr      ]
iconTooltip  [textures/test.texr      ]
fontRegular  [fonts/default.ttf       ]
```
- `ImGui::InputText` (max 256 chars)

**Prévisualisation :**
Bloc ImGui style-appliqué simulant un panneau de jeu :
```
┌─ Preview ─────────────────────────────────────┐
│  [Titre 20px]                                  │
│  Corps de texte 14px (muted)                   │
│  [Bouton Primary]   [Bouton Secondary]         │
│  ░░░░░░░░░░░░░░░░ surface / panel border       │
└────────────────────────────────────────────────┘
```
Dessiné avec `ImGui::PushStyleColor` / `ImGui::PopStyleColor` appliquant les couleurs du thème sélectionné.

Chaque modification positionne `m_dirty = true`.

### `engine/editor/EditorMode.h`

Ajouter :
```cpp
#include "engine/editor/EditorThemePanel.h"
engine::editor::EditorThemePanel m_themePanel;
```

### `engine/editor/EditorMode.cpp`

- Dans `Init()` : `m_themePanel.Load(config);`
- Rendu ImGui :
  ```cpp
  if (ImGui::Begin("Theme Editor"))
      m_themePanel.Draw();
  ImGui::End();
  ```
- Propager `m_themePanel.IsDirty()` dans `m_dirty`
- `Ctrl+S` → `m_themePanel.Save(config)` quand la fenêtre Theme est active
