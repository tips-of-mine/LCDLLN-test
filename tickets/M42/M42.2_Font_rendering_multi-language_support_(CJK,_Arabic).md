
**Objectif**
Support rendu fonts CJK (Chinese, Japanese, Korean) et RTL (Arabic, Hebrew).

**Dépendances**
- M42.1
- M16.1

**Livrables**
- Font atlas génération (support Unicode)
- CJK glyph rendering (large character sets)
- RTL text layout (reverse order)
- Font fallback (multiple fonts per locale)

**Structure & chemins (verrouillé)**
- **Code moteur** : uniquement sous `/engine/...`
- **Contenu** (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- **Outils offline** : uniquement sous `/tools/...`
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

**Spécification technique**
- Font library: FreeType2 (rasterize glyphs)
- Atlas: dynamic (add glyphs on-demand) ou pre-baked
- CJK: large atlas (4096x4096 ou multiple pages)
- RTL: reverse text layout (bidi algorithm)

**Étapes d'implémentation**
1. Integrate FreeType2
2. Load font file (.ttf): per locale (Noto Sans, Noto Sans CJK)
3. Generate glyph atlas:
   - Latin: pre-bake common chars
   - CJK: dynamic (add glyph on first use)
4. Render text:
   - Measure glyphs, layout (LTR ou RTL)
   - Draw quads with atlas UVs
5. RTL: detect Arabic/Hebrew, reverse string
6. Font fallback: if glyph missing, try fallback font

**Critères de validation (DoD)**
- [ ] English text renders
- [ ] Chinese characters render
- [ ] Arabic text reverse correctement
- [ ] Emoji support (bonus)

**Notes / pièges à éviter**
- Atlas size limit: multiple textures si needed
- Bidi: complex (use ICU library helpers)
