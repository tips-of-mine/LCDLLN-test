# M09.9 — ChunkBudgetStats : LogStats basé sur temps réel (pas sur le frame count)

## Objectif
Remplacer le déclenchement de `LogStats` toutes les 60 frames par un timer en
secondes réelles, afin d'obtenir un monitoring exact à 1 seconde d'intervalle,
indépendamment du framerate.

## Contexte
Dans `Engine.cpp` :
```cpp
if (m_currentFrame % 60 == 0)
    m_chunkStats.LogStats();
```
À 120 fps → log toutes les 0.5 s. À 30 fps → log toutes les 2 s.
Le monitoring des budgets perd toute sa signification dès que le framerate dévie de 60.

## Dépendances
- M09.2 (ChunkBudgetStats existant)
- M00.1 (Time system — `Time::TotalSeconds()`)

## Livrables
- `engine/world/ChunkBudgetStats.h`
  - Membre privé `double m_lastLogTime = -1.0`.
  - Méthode `void LogStatsIfDue(double nowSeconds, double intervalSeconds = 1.0)`.
- `engine/world/ChunkBudgetStats.cpp`
  - `LogStatsIfDue` : si `nowSeconds - m_lastLogTime >= intervalSeconds`, appeler `LogStats()` et mettre à jour `m_lastLogTime`.
  - Initialiser `m_lastLogTime = -intervalSeconds` dans `Init` (pour logguer dès la première seconde).
  - Lire `world.budget_log_interval_s` dans `Init` (config, défaut 1.0, min 0.1).
- `engine/Engine.cpp`
  - Remplacer `if (m_currentFrame % 60 == 0) m_chunkStats.LogStats()` par `m_chunkStats.LogStatsIfDue(m_time.TotalSeconds())`.

## Spécification technique
```cpp
void ChunkBudgetStats::LogStatsIfDue(double nowSeconds, double intervalSeconds)
{
    if (m_lastLogTime < 0.0)
        m_lastLogTime = nowSeconds - intervalSeconds; // force premier log

    if (nowSeconds - m_lastLogTime >= intervalSeconds)
    {
        LogStats();
        m_lastLogTime = nowSeconds;
    }
}
```
- `nowSeconds` = `m_time.TotalSeconds()` (steady_clock depuis le boot, déjà disponible dans M00.1).
- Passer `nowSeconds` en paramètre facilite les tests unitaires (pas de dépendance `std::chrono` directe dans `ChunkBudgetStats`).
- Ne pas modifier l'intervalle en runtime (la config est lue au boot).

## Étapes d'implémentation
1. Vérifier que `engine::core::Time` expose `TotalSeconds() const -> double`.
2. Ajouter `m_lastLogTime` et `LogStatsIfDue` dans `.h`.
3. Implémenter `LogStatsIfDue` dans `.cpp`.
4. Mettre à jour `Init` pour lire `world.budget_log_interval_s`.
5. Remplacer le call-site dans `Engine.cpp`.

## Notes / pièges à éviter
- Ne pas utiliser `std::chrono` directement dans `ChunkBudgetStats` : injecter `nowSeconds` depuis l'appelant.
- `m_lastLogTime = -1.0` est un sentinel ; initialiser à `-intervalSeconds` est plus propre.
- Ne pas supprimer `LogStats()` public : il reste utilisable pour debug forcé.
