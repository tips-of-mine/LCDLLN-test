# M37.1_Water_plane_mesh_+_reflection refraction

## Objectif
Rendu eau avec plane mesh, reflections, refractions, Fresnel.

## Dépendances
- M03.2
- M08.2

## Livrables
- Water plane mesh (tiled)
- Reflection: render scene mirrored
- Refraction: render underwater scene
- Fresnel blend (edge transparency)

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Water plane: Y=constant (lake) ou mesh (river)
- Reflection pass: flip camera Y, render above-water
- Refraction pass (optional): render below-water (optional distortion)
- Fresnel: F = F0 + (1-F0)*(1 - dot(N,V))^5
- Normal map: animé (scroll UV)

## Étapes d'implémentation
1. Water mesh: create plane (or load mesh)
2. Reflection pass: mirror camera, render to ReflectionRT
3. Refraction pass (optional): render underwater to RefractionRT
4. Water shader:
   - Sample reflection/refraction textures
   - Blend via Fresnel
   - Normal map distortion (ripples)
   - Depth fade (shallow=transparent, deep=opaque)
5. Blend HDR SceneColor

## Rapport final (M37.1)

### 1) FICHIERS

**Créés :**
- `engine/render/WaterPass.h` — classe `WaterPass` : water plane mesh (32×32 grid, HOST_VISIBLE), reflection RT + refraction RT (DEVICE_LOCAL, R16G16B16A16_SFLOAT, half-res), pipeline alpha-blend composite sur SceneColorHDR, push constants `WaterParams` (96 bytes), helper statique `BuildReflectionViewProj()`.
- `engine/render/WaterPass.cpp` — implémentation complète en raw Vulkan (`vkCreateBuffer/vkAllocateMemory/vkBindBufferMemory` + `vkCreateImage/vkAllocateMemory/vkBindImageMemory`) ; Destroy() en ordre inverse ; LOG_INFO/WARN/ERROR sur tous les chemins.
- `game/data/shaders/water.vert` — Vertex shader GLSL : étend la grille [0,1]² en world-space (±planeHalfSize, waterY).
- `game/data/shaders/water.frag` — Fragment shader GLSL : échantillonnage reflection/refraction, distortion via normal procedurale animée (dual-layer scroll), Fresnel Schlick `F = F0 + (1-F0)*(1 - dot(N,V))^5`, tinte deep-water, alpha selon angle.
- `tickets/issues/M37.1_Water_plane_mesh_+_reflection_refraction_Issue.md`

**Modifiés :**
- `engine/render/DeferredPipeline.h` — `#include WaterPass.h`, membre `WaterPass m_waterPass`, getter `GetWaterPass()`.
- `engine/render/DeferredPipeline.cpp` — Init du WaterPass après TAA (non-fatal, warn si shaders absents) ; Destroy() inclut `m_waterPass.Destroy(device)` ; lit `render.water.reflection_width/height` depuis config (défaut 512×256).
- `CMakeLists.txt` — `engine/render/WaterPass.cpp` ajouté à la target `engine_app`.

### 2) COMMANDES WINDOWS À EXÉCUTER
