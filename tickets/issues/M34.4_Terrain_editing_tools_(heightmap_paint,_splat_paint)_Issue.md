# M34.4_Terrain_editing_tools_(heightmap_paint,_splat_paint).md

## Objectif
Outils éditeur pour modifier heightmap et splat maps runtime ou offline.

## Dépendances
- M34.2
- M12.1

## Livrables
- Brush tool (raise, lower, smooth, flatten)
- Splat paint tool (paint layers)
- Realtime update terrain mesh
- Save/export modified maps

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Brush: radius, strength, falloff
- Operations: raise, lower, smooth (gaussian), flatten (target height)
- Splat paint: select layer, paint weight
- Update: modify texture CPU, upload GPU

## Étapes d'implémentation
1. UI: brush settings (size, strength, operation)
2. Raycast mouse->terrain (get edit position)
3. Apply brush kernel (circular falloff)
4. Heightmap: modify pixels in radius
5. Upload dirty region texture->GPU
6. Regenerate affected patches mesh (optional, ou shader reads texture)
7. Splat paint: similar, modify splat map RGBA
8. Save: export heightmap/splat to file
