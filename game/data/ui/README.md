# assets/ui/

🇫🇷 FRANÇAIS
## Pourquoi ce dossier existe
Il regroupe toutes les images utilisées par les différents écrans de l’interface utilisateur.

## Ce qu’il doit contenir
- `common/` : images partagées entre plusieurs écrans.
- `login/` : images de l’écran de connexion.
- `register/` : images de l’écran de création de compte.
- `server_list/` : images de l’écran de sélection des serveurs.
- `dialogs/` : images pour les fenêtres de dialogue.

---

🇬🇧 ENGLISH
## Why this folder exists
It groups all images used by the different UI screens.

## What it must contain
- `common/` : shared images.
- `login/` : login screen images.
- `register/` : account creation screen images.
- `server_list/` : server selection screen images.
- `dialogs/` : dialog and popup images.
