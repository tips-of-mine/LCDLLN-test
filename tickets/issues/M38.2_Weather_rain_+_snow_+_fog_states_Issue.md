# M38.2_Weather_rain_+_snow_+_fog_states

## Objectif
Système météo avec états (clear, rain, snow, fog) et transitions.

## Dépendances
- M38.1
- M17.2

## Livrables
- Weather states (enum)
- Transition system (fade in/out)
- Rain particles (vertical streaks)
- Snow particles (falling flakes)
- Fog density (distance-based)

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Weather states: Clear, Rain, Snow, Fog, Storm
- Duration: 5-20min par state
- Transition: 30s fade
- Rain: particle emitter (vertical, fast)
- Snow: particle emitter (slow, lateral drift)
- Fog: exponential fog density

## Étapes d'implémentation
1. Weather system: current state, timer
2. Transition: lerp intensity (0->1 over 30s)
3. Rain particles: spawn rate 1000/sec, lifetime 2s
4. Snow particles: spawn rate 500/sec, lifetime 5s
5. Fog: set fog density param (distance/color)
6. Audio: rain sound loop (volume = intensity)
7. Surface wetness: shader darkening (optional)

## Notes / pièges à éviter
