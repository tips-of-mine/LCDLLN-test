
**Objectif**
Implémenter character controller avec collision capsule, gravity, slope handling.

**Dépendances**
- M0.4 (input)
- M9.1 (world)

**Livrables**
- `engine/gameplay/CharacterController.*`
- Capsule collision shape
- Movement avec acceleration/friction
- Slope/step handling

**Structure & chemins (verrouillé)**
- **Code moteur** : uniquement sous `/engine/...`
- **Contenu** (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- **Outils offline** : uniquement sous `/tools/...`
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

**Spécification technique**
- Capsule (rayon 0.3m, hauteur 1.8m typique)
- Walk speed 4-6 m/s, run 8-10 m/s
- Gravity -20 m/s²
- Max slope 45°, max step 0.3m

**Étapes d'implémentation**
1. Définir CharacterController (velocity, isGrounded, capsule dims)
2. Update: appliquer input -> desired velocity
3. Appliquer gravity si !isGrounded
4. Collision detection capsule vs world (sweep test)
5. Slide along obstacles (project velocity)
6. Step up detection

**Critères de validation (DoD)**
- [ ] WASD déplace personnage sans passer à travers murs
- [ ] Gravity appliquée, personnage tombe
- [ ] Monte escaliers <30cm automatiquement
- [ ] Bloqué par pentes >45°

**Notes / pièges à éviter**
- Utiliser sweep test, pas discrete collision
- Séparer logique movement de physics (fixed timestep optionnel)
