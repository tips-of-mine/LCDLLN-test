# M25.4_Animation_IK_(Inverse_Kinematics)_feet_placement

## Objectif
Ajuster pieds pour coller au terrain (foot IK) pour éviter flottement.

## Dépendances
- M25.2
- M09.1 (terrain/world)

## Livrables
- Two-bone IK solver (jambe)
- Raycast terrain par pied
- Ajustement pose finale

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Raycast depuis position pied animation vers terrain
- Two-bone IK: résoudre hip->knee->foot
- Limiter offset vertical (max 20-30cm)

## Étapes d'implémentation
1. Identifier joints pied/genou/hanche dans skeleton
2. Après playback animation: raycast par pied
3. Calculer target foot position (hit terrain)
4. Appliquer two-bone IK solver
5. Optionnel: ajuster hips hauteur si offset trop grand
