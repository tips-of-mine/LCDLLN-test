# M38.1_Day-night_cycle_sun_position_+_sky_gradient

## Objectif
Cycle jour/nuit avec soleil/lune animés, sky gradient, éclairage dynamique.

## Dépendances
- M04.1
- M05.2

## Livrables
- Time of day (0-24h, speed configurable)
- Sun/moon position (celestial mechanics)
- Sky gradient (Rayleigh scattering approximation)
- Directional light rotation + color/intensity

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Time: 0-24h, update dt * timeScale (1 = realtime, 60 = 1h realtime = 1day ingame)
- Sun position: latitude/longitude -> azimuth/elevation
- Sky colors: dawn=orange, noon=blue, dusk=red, night=dark blue
- Light intensity: day=1.0, night=0.1 (moonlight)

## Étapes d'implémentation
1. Time system: current time (float 0-24), advance each frame
2. Calculate sun angle: elevation = sin(time * pi / 12 - pi/2)
3. Sun direction: rotate directional light
4. Sky shader: gradient top->horizon (Rayleigh scatter approximation)
5. Light color: lerp white->orange (sunrise/set), white (noon)
6. Ambient light: scale by sun elevation
7. Moon: opposite side (sun + 12h)

### 1) FICHIERS

**Créés :**
- `engine/render/DayNightCycle.h` — classe `DayNightCycle` avec struct `SkyState` (timeHours, sunElevation, sunDirection, moonDirection, lightColor, lightIntensity, ambientColor, skyZenith, skyHorizon), méthodes `Init()`, `Update(dt)`, `SetTime()`, `ToAtmosphereSettings()`.
- `engine/render/DayNightCycle.cpp` — implémentation complète :
  - Avancement du temps : `timeHours += dt * timeScale / 3600`
  - Élévation soleil (spec) : `elevation = sin(t * π/12 - π/2)`
  - Direction soleil : vecteur normalisé (cosElev, elevation, 0)
  - Moon : direction opposée au soleil
  - Couleurs ciel : dawn=orange, noon=blue, dusk=red, night=darkblue via lerps smoothstep
  - Intensité lumineuse : 1.0 jour → 0.1 nuit (plancher 0.1 pour éviter le noir total)
  - `ToAtmosphereSettings()` : production de `AtmosphereSettings` compatible avec `m_zoneAtmosphere`
- `game/data/shaders/sky.vert` — vertex shader fullscreen triangle, reconstruction de ray via `invViewProj`
- `game/data/shaders/sky.frag` — sky gradient Rayleigh approx, disque solaire/lunaire, étoiles nocturnes, haze horizon
