### Détail implémentation

- **ChunkRequest** : chunkId (ChunkCoord), targetState (RingType).
- **QueuedChunkRequest** : ChunkRequest + enqueueTime pour l’aging.
- **ComputeChunkPriority** : base par ring (Active 1000, Visible 100, Far 1) + facteur direction (dot vue→chunk avec viewDir, ±50) + boost vieillissement (config streaming.aging_boost_max).
- **StreamingScheduler** : request queue = vector de QueuedChunkRequest, tri par priorité au PopNextRequest ; io/cpu/gpuUpload = std::queue&lt;ChunkRequest&gt; avec Push/Pop.
- **Producer** : dans Engine::Update, après mise à jour caméra, viewDir XZ dérivé de yaw/pitch, EmitChunkRequestsForPosition(playerX, playerZ, callback) ; le callback push chaque chunk avec PushRequest(..., currentTime).
