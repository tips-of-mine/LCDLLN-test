
**Objectif**
Permettre joueur repositionner/redimensionner éléments HUD.

**Dépendances**
- M16.2 (HUD elements)

**Livrables**
- Edit mode: unlock HUD elements
- Drag & drop reposition
- Resize handles (corners)
- Save layout per character
- Presets (default, minimal, compact)

**Structure & chemins (verrouillé)**
- **Code moteur** : uniquement sous `/engine/...`
- **Contenu** (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- **Outils offline** : uniquement sous `/tools/...`
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

**Spécification technique**
- Elements: player frame, target frame, minimap, action bars, chat
- Position: (x, y) % screen
- Scale: 0.5-2.0x
- Lock: prevent accidental moves

**Étapes d'implémentation**
1. Edit mode toggle: ESC -> Interface -> Edit Layout
2. Show borders around all HUD elements
3. Drag: mouse down -> move widget
4. Resize: corners show handles, drag to scale
5. Snap to grid (optional): 10px increments
6. Save layout: JSON (elementId, x, y, scale)
7. Load layout: apply positions on login
8. Presets: load predefined layouts (buttons)

**Critères de validation (DoD)**
- [ ] Edit mode affiche borders
- [ ] Drag reposition fonctionne
- [ ] Resize change taille
- [ ] Save persist layout
- [ ] Presets chargent configs

**Notes / pièges à éviter**
- Reset button (restore defaults)
- Limit scale (min 0.5, max 2.0)
