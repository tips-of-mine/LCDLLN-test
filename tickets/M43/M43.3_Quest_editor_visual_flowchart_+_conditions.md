
**Objectif**
Éditeur quêtes visuel avec flowchart steps, conditions, rewards.

**Dépendances**
- M15.1 (quests)
- M12.1

**Livrables**
- Quest editor UI (flowchart nodes)
- Steps: talk NPC, kill mobs, collect items, enter area
- Conditions: level requirement, prereq quests
- Rewards: XP, gold, items
- Save/load quest JSON

**Structure & chemins (verrouillé)**
- **Code moteur** : uniquement sous `/engine/...`
- **Contenu** (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- **Outils offline** : uniquement sous `/tools/...`
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

**Spécification technique**
- Nodes: start, step, choice, end
- Steps: typed (kill, collect, talk, explore)
- Conditions: AND/OR logic (level, quest complete, item owned)
- Rewards: list (XP, gold, items)

**Étapes d'implémentation**
1. UI: flowchart canvas
2. Create nodes: drag from palette
3. Connect: draw arrows step->step
4. Node properties panel:
   - Step type dropdown
   - Parameters (e.g., kill 10 wolves)
   - Conditions list
5. Validate: start node exists, all steps connected
6. Save: serialize graph to JSON (M15.1 format)
7. Load: parse JSON, build graph UI

**Critères de validation (DoD)**
- [ ] Create quest avec 3 steps
- [ ] Connect steps fonctionne
- [ ] Save JSON valide
- [ ] Load quest fonctionne

**Notes / pièges à éviter**
- Localization: quest text keys (not hardcoded)
- Validation: cycles detection (infinite loops)
