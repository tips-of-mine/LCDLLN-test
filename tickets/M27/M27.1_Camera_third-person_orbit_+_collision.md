
**Objectif**
Caméra third-person standard MMO avec orbite autour personnage et collision environnement.

**Dépendances**
- M0.4 (input mouse)
- M26.1 (character)

**Livrables**
- `engine/render/Camera.*` (view/proj)
- `engine/gameplay/ThirdPersonCamera.*`
- Collision camera vs world (raycast/sweep)

**Structure & chemins (verrouillé)**
- **Code moteur** : uniquement sous `/engine/...`
- **Contenu** (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- **Outils offline** : uniquement sous `/tools/...`
- Tous les chemins de contenu sont **relatifs** à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

**Spécification technique**
- Distance default 5m, zoom 2-10m
- Pitch limits -30° à +80° (éviter gimbal lock)
- Yaw 360° libre
- Collision: rapproche caméra si obstruction

**Étapes d'implémentation**
1. Track target position (character position + offset up)
2. Input mouse: delta -> rotate yaw/pitch
3. Calculer position cam: target + offset spherical (yaw, pitch, distance)
4. Raycast target->cam position: si hit, clamp distance
5. Smooth interpolation (spring damping)

**Critères de validation (DoD)**
- [ ] Caméra orbit autour personnage
- [ ] Collision rapproche camera smooth
- [ ] Zoom molette fonctionne
- [ ] Pas de clipping à travers murs

**Notes / pièges à éviter**
- Sphere cast plutôt que raycast pour éviter jitter
- Fade character si trop proche (shader alpha)
