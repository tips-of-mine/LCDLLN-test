# M26.3_Character_controller_swimming_+_flying_(optional)

## Objectif
Support mouvement aquatique et vol (races volantes optionnel).

## Dépendances
- M26.1
- M37.1

 ## Livrables
- Movement mode: ground/air/water
- Swimming physics (buoyancy)
- Flying physics (optional)

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

 ## Spécification technique
- Swimming: reduced gravity, 3D movement, surface/dive
- Flying: full 3D, no gravity, stamina-based (optional)
- Transition smooth entre modes

 ## Étapes d'implémentation
1. Detect water volume entry/exit
2. Swimming mode: velocity damping, vertical control (space/ctrl)
3. Surface breaching: force upward near surface
4. Flying mode (si activé): pitch/yaw control, stamina drain
5. Brancher animations swim/fly

## Definition of Done (DoD)
- [ ] Build Windows OK (configure + build) via presets existants
- [ ] Exécutable lancé (smoke test) et pas de crash immédiat
- [ ] Aucun dossier racine non autorisé créé
- [ ] Rapport final: fichiers modifiés + commandes + résultats + DoD
