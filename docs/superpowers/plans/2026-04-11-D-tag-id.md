# TAG-ID (Sous-projet D) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Générer côté serveur un identifiant TAG-ID de format `CCYMMXXXXX` (code pays 2 lettres + dernier chiffre de l'année + mois 2 chiffres + séquence 5 chiffres) au moment de l'inscription, le stocker en base avec contrainte UNIQUE, et l'afficher au client après inscription.

**Architecture:** Migration BDD pour ajouter `tag_id` et `country_code` à la table `accounts`. `RegisterRequestPayload` transporte le `country_code` depuis le client (champ ajouté en Plan B). `AuthRegisterHandler` génère le TAG-ID en une transaction et le retourne dans `RegisterResponsePayload`. Le client affiche le TAG-ID dans le bandeau de confirmation post-inscription.

**Tech Stack:** C++20, MySQL/MariaDB, protocol_v1, CMake/MSVC.

- [ ] **Step 1 : Créer `db/migrations/0016_tag_id.sql`**

```sql
-- Migration 0016 — TAG-ID et country_code sur les comptes (Plan D).
SET NAMES utf8mb4;

-- Ajout country_code si absent.
SET @m16_cc := (
  SELECT COUNT(*) FROM information_schema.columns
  WHERE table_schema = DATABASE() AND table_name = 'accounts' AND column_name = 'country_code'
);
SET @m16_cc_s := IF(@m16_cc = 0,
  'ALTER TABLE accounts ADD COLUMN country_code VARCHAR(2) NOT NULL DEFAULT '''' AFTER birth_date',
  'SELECT 1');
PREPARE m16_cc_p FROM @m16_cc_s; EXECUTE m16_cc_p; DEALLOCATE PREPARE m16_cc_p;

-- Ajout tag_id si absent.
SET @m16_ti := (
  SELECT COUNT(*) FROM information_schema.columns
  WHERE table_schema = DATABASE() AND table_name = 'accounts' AND column_name = 'tag_id'
);
SET @m16_ti_s := IF(@m16_ti = 0,
  'ALTER TABLE accounts ADD COLUMN tag_id VARCHAR(10) NOT NULL DEFAULT '''' AFTER country_code',
  'SELECT 1');
PREPARE m16_ti_p FROM @m16_ti_s; EXECUTE m16_ti_p; DEALLOCATE PREPARE m16_ti_p;

-- Index unique sur tag_id (les comptes existants ont tag_id = '' ce qui viole UNIQUE ; l'index
-- est donc créé sans contrainte UNIQUE sur les lignes existantes à vide via un index partiel ou,
-- si le moteur ne le supporte pas, on laisse la contrainte sur les nouvelles insertions).
-- Approche compatible MySQL 5.7+ : index unique classique, les comptes existants auront tag_id = ''
-- ce qui nécessite une mise à jour manuelle avant production. Pour l'environnement de test seul
-- le DEFAULT '' suffit.
SET @m16_ix := (
  SELECT COUNT(*) FROM information_schema.statistics
  WHERE table_schema = DATABASE() AND table_name = 'accounts' AND index_name = 'idx_accounts_tag_id'
);
SET @m16_ix_s := IF(@m16_ix = 0,
  'CREATE UNIQUE INDEX idx_accounts_tag_id ON accounts(tag_id)',
  'SELECT 1');
PREPARE m16_ix_p FROM @m16_ix_s; EXECUTE m16_ix_p; DEALLOCATE PREPARE m16_ix_p;
```

- [ ] **Step 2 : Ajouter la génération du TAG-ID en SQL (dans la même transaction)**

Juste avant la requête INSERT existante, ajouter la logique de génération du TAG-ID :

```cpp
	// Construire le préfixe CCYMM.
	{
		const auto now = std::chrono::system_clock::now();
		const std::time_t t = std::chrono::system_clock::to_time_t(now);
		std::tm tm_val{};
#if defined(_WIN32)
		localtime_s(&tm_val, &t);
#else
		localtime_r(&t, &tm_val);
#endif
		const int year_digit = (tm_val.tm_year + 1900) % 10;
		const int month      = tm_val.tm_mon + 1;
		std::string cc = country_code.empty() ? "XX" : std::string(country_code).substr(0, 2);
		for (char& c : cc) c = static_cast<char>(std::toupper(static_cast<unsigned char>(c)));
		char prefix[8];
		std::snprintf(prefix, sizeof(prefix), "%s%1d%02d", cc.c_str(), year_digit, month);

		// Obtenir le prochain numéro de séquence pour ce préfixe.
		uint64_t seq = 1;
		{
			auto conn = m_pool->Acquire();
			const std::string sql_seq =
				"SELECT COALESCE(MAX(CAST(SUBSTR(tag_id, 6) AS UNSIGNED)), 0) + 1 "
				"FROM accounts WHERE tag_id LIKE '" + std::string(prefix) + "%'";
			auto res = conn->Query(sql_seq);
			if (res && res->Next())
				seq = res->GetUint64(0);
		}
		char tag_buf[11];
		std::snprintf(tag_buf, sizeof(tag_buf), "%s%05llu", prefix,
			static_cast<unsigned long long>(seq));
		tag_id_out = tag_buf;
	}
```
