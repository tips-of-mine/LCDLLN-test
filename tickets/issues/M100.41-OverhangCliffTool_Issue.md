# M100.41 — Overhang Cliff Tool (Phase 11 — Volumes 3D)

## Résumé

Outil spécialisé pour poser des **falaises avec surplomb** : meshes glTF qui simulent une roche projetée horizontalement au-dessus d'un point bas. Contrairement à la heightmap 2D (incapable de représenter un overhang), un mesh insert résout le problème en ajoutant un volume 3D solide au-dessus du terrain.

Réutilise intégralement le `MeshInsertSystem` de M100.40 (fondation : data model, sérialisation, streaming, gizmo, undo) et n'ajoute que :

- **Catalogue d'overhangs** : assets glTF spécialisés (surplomb subtil, surplomb dramatique, balcon rocheux, corniche large…), au moins 4 entrées MVP.
- **Anchoring vertical** : la face arrière du mesh (face « scellée » sur la falaise existante) est alignée à la **normale moyenne du terrain** sur une bande de 3 m de hauteur, pas seulement au sol. Le mesh « pousse hors » d'une pente verticale ou raide.
- **Auto-détection de pente** : le clic ne se fait pas n'importe où sur le terrain : l'outil **filtre les positions valides** (pente locale > 60° préférée, refuse les zones plates avec un cursor rouge).
- **Camouflage spécifique** : pas de patch splat sable/rocher comme pour les grottes, mais une **bande de splat** sur la falaise verticale pour fondre la jointure mesh ↔ terrain.

Catégorie `insertCategory = "overhang"` dans `MeshInsertInstance`.

**Aucun nouvel exécutable.** Code dans `engine_core`.

#### Partie D — Anchoring vertical

Quand le curseur survole une cellule du terrain, l'outil :

1. Sample la **normale terrain** sur un voisinage 5×5 cells autour du point (moyennée).
2. Calcule la pente : `slopeDeg = acos(normal.y) * 180 / π`.
3. Si `slopeDeg < minTerrainSlopeDeg` du catalogue → cursor **rouge**, click ignoré.
4. Sinon → cursor **vert**, preview ghost affiché aligné :
   - `worldPosition` = position du clic sur le terrain.
   - `rotationY` calculé pour que la face arrière du mesh soit alignée avec le plan tangent terrain (`anchorPlaneNormal` rotaté vers `terrainNormal.xz`).
   - `offsetZ` (saillie) pousse le mesh hors de la falaise dans la direction `-anchorPlaneNormal`.

#### Partie E — Camouflage jointure

Au lieu du patch splat circulaire (cave M100.40), on peint une **bande verticale** :

- Définie en plan tangent à la falaise.
- Largeur et hauteur configurables.
- Couche splat configurable (typ. `Rock`).
- Force progressive (1.0 au centre de la bande, 0 aux bords) via smoothstep.

Algorithmiquement : pour chaque cellule du terrain dans une AABB locale autour du point d'ancrage, calcule la projection sur le plan tangent et applique le poids splat.

Pas de props auto (les overhangs n'ont pas typiquement besoin de rochers de camouflage).
