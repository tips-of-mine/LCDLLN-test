# M34.3_Terrain_holes_+_cliffs_+_overhangs_(optional)

## Objectif
Support trous terrain (caves entrances) et overhangs (non-heightmap zones).

## Dépendances
- M34.1

## Livrables
- Hole mask texture (1-bit per quad)
- Cliff meshes (custom geometry)
- Blend heightmap + custom meshes

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Hole mask: R8 texture (0=hole, 255=solid)
- Discard pixels si hole (shader)
- Cliffs: separate static meshes (blend edges)

## Étapes d'implémentation
1. Hole mask texture (same res que heightmap)
2. Shader: sample mask, discard if hole
3. Collision: mark holes in navmesh (unwalkable)
4. Cliff meshes: author manuellement, place in world
5. Blend edges: duplicate verts, match heightmap border

- Overhangs = full custom mesh (pas heightmap)
- Limit holes (performance impact)
