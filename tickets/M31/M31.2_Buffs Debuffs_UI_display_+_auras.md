# M31.2_Buffs Debuffs_UI_display_+_auras

## Objectif
Afficher buffs/debuffs UI (icônes, timers) et auras visuelles (glow, particles).

## Dépendances
- M31.1
- M16.1
- M17.2

## Livrables
- Buff bar UI (player + target)
- Icon + duration timer
- Visual auras (particle effects, model glow)

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Buff bar: horizontal row icons (top-right player)
- Debuff bar: separate (different color border)
- Timer: cooldown sweep ou text countdown
- Auras: persistent particle emitter (heal=green, poison=purple)

## Étapes d'implémentation
1. UI: create buff/debuff icon widgets
2. Subscribe à StatusEffect events (add, remove, update)
3. Display icon + stack count + timer
4. Sort: buffs left, debuffs right (ou top/bottom)
5. Visual FX: spawn particle emitter at character root
6. Color model: shader tint (stun=yellow glow, poison=green)
7. Destroy FX on effect expire

## Definition of Done (DoD)
- [ ] Build Windows OK (configure + build) via presets existants
- [ ] Exécutable lancé (smoke test) et pas de crash immédiat
- [ ] Aucun dossier racine non autorisé créé
- [ ] Rapport final: fichiers modifiés + commandes + résultats + DoD

## Notes / pièges à éviter
- Limit icons displayed (show 10 max, prioritize important)
- Tooltip on hover (effect description)
