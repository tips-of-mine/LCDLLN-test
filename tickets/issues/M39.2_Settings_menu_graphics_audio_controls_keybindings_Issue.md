## Contenu du ticket

# M39.2_Settings_menu_graphics audio controls keybindings

## Objectif
Menu settings complet avec onglets graphics, audio, controls, gameplay.

## Dépendances
- M16.1
- M20.2

## Livrables
- Settings UI panel (tabs)
- Graphics: resolution, quality preset, vsync, FOV
- Audio: master/sfx/music/voice volumes, device selection
- Controls: mouse sensitivity, invert Y, keybindings
- Gameplay: auto-loot, combat text, camera distance

## Structure & chemins (verrouillé)
- Code moteur : uniquement sous `/engine/...`
- Contenu (textures/meshes/json/audio) : uniquement sous `/game/data/...`
- Outils offline : uniquement sous `/tools/...`
- Tous les chemins de contenu sont relatifs à `paths.content` (config.json).
- ❌ Interdit : créer un dossier racine `/assets` ou utiliser des chemins absolus.

## Spécification technique
- Persist settings: local config file + server-side (cloud save)
- Apply changes: realtime ou restart required (label)
- Keybindings: rebindable, detect conflicts

## Étapes d'implémentation
1. UI: tabbed panel (graphics, audio, controls, gameplay)
2. Graphics tab: Resolution dropdown, Quality preset, vsync/fullscreen, FOV slider (60-120)
3. Audio tab: Sliders master/sfx/music (0-100%), output device dropdown
4. Controls tab: Mouse sensitivity slider, invert Y checkbox, keybindings list
5. Gameplay tab: auto-loot, combat text, timestamps, camera distance (2-10m)
6. Save settings: write config.json (local), upload to server (cloud)
7. Apply: update engine params, reload if needed

## Definition of Done (DoD)

- [ ] Aucun dossier racine non autorisé créé
- [ ] Rapport final: fichiers modifiés + commandes + résultats + DoD

## Notes / pièges à éviter
- Default keybinds always accessible (reset button)
- Validate settings (e.g., unsupported resolution)
